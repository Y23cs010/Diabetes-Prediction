# Diabetes Prediction using Machine Learning

## Project Overview
This project predicts whether a person has diabetes using a Random Forest Classifier.

## Dataset
Pima Indians Diabetes Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Steps
1. Load Dataset
2. Data Preprocessing
3. Split Data
4. Train Model
5. Test Model
6. Predict Diabetes

## Accuracy
72%

## How to Run

```bash
pip install -r requirements.txt
```

Run the notebook in Google Colab or Jupyter Notebook.
